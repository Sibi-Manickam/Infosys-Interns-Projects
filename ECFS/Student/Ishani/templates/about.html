Data Preprocessing

Library Importation: Essential libraries such as Pandas for data manipulation, Seaborn for visualization, and Matplotlib for plotting were imported.

Data Loading: The dataset 'Statewise Consumption.csv' was loaded into the environment. It contains the following attributes:

  • States
  • Regions
  • Latitude
  • Longitude
  • Dates
  • Usage

Data Cleaning:

  • Missing Values Handling: Any missing values in the dataset were addressed.
  • Duplicate Removal: Duplicated entries within the dataset were identified and removed.

Feature Engineering

Temporal Feature Extraction: The 'Dates' column was processed to derive additional features like year, month, quarter, and day of the week.

Categorical Encoding: Categorical features were encoded using one-hot encoding for further analysis.

Exploratory Data Analysis (EDA):

  • Usage Distribution Analysis:

    • Statewise usage distribution over time was analyzed to identify trends and patterns in electricity consumption.

  • Statewise Usage Analysis:

    • Monthly, quarterly, and yearly usage patterns were explored for each state individually.

  • Consumption Variation Analysis:

    • Variations in consumption across different regions on yearly basis were examined to identify trends and patterns.

  • Seasonal Consumption Patterns:

    • Seasonal consumption patterns were investigated to understand how usage varies throughout the year.

  • Day of the Week Analysis:

    • The impact of the day of the week on consumption variation was analyzed to uncover any significant trends.

Model Building and Evaluation

Prophet Model Utilization: The Prophet model was employed for electricity usage forecasting, providing insights into future consumption for each state.

Prophet Model Evaluation: Mean Absolute Error (MAE) and Root Mean Squared Error (RMSE) were calculated to assess the performance of the Prophet model.

LSTM Model Implementation: Long Short-Term Memory (LSTM) models were developed to forecast usage, offering an alternative approach to forecasting.

LSTM Model Evaluation: The performance of the LSTM models was evaluated using MAE and RMSE metrics, providing insights into their accuracy and reliability.
